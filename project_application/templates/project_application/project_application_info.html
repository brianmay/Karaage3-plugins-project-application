Note

Project applications are managed by the e-mail address that you provided during the application process.
Institutional e-mail addresses are required.

Every project must have a Project Supervisor*.
It is recommended that the applicant is the Project Supervisor.

Students can not be the Project Supervisor

* The Project Supervisor (Chief Principle Investigator) is responsible for the larger scientific context of the work and generally hold the overarching funding grant of the project. They will be responsible for all resource usage, including data management. All compute, storage and software resources are accounted and reported to the Supervisor's Host institute.
